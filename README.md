# ecoleta

## About

ecoleta is a system made for registering garbage recycable points. It was made in Rocketseat Next Level Week.

## Tecnologies

It was used:

* html, css and javascript
* Node.js
  * nunjucks
  * SQlite

## Running

Run the following commands to start the server at `localhost:3000`:

```bash
$> npm install
$> npm start
```
